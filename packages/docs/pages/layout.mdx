# Layout, Container and Grid

Any well-designed project should be done in a grid. This chapter describes how to code grid in your project.

## `Container`

Main element is `Container`. It doesn't do much, just creates a container with margins.

```javascript live=true fullscreen=true
<Container>
  <Box sx={{ bg: "mono300", p: 3 }}>Lorem ipsum dolor sit amet</Box>
</Container>
```

By default `Container` has `10vw` margin.

You can change default in a theme by setting `containers.main` value:

```javascript
// theme.js

export default {
  containers: {
    main: ["10px", null, "10vw"],
    secondary: ["30vw", null, null, "20vw"]
  }
  // remaining config...
};
```

As you see, apart from `main` we also defined `secondary` container. We can use it this way:

```javascript live=true fullscreen=true
<Container margin={"secondary"}>
  <Box sx={{ bg: "mono300", p: 3 }}>Lorem ipsum dolor sit amet</Box>
</Container>
```

If you want to contextually create a different `Container` and not put it into a theme, this is how you do it:

```javascript live=true fullscreen=true
<Container margin={["20px", "40px", "60px", "80px", "100px"]}>
  <Box sx={{ bg: "mono300", p: 3 }}>Lorem ipsum dolor sit amet</Box>
</Container>
```

Frequently you want to zero margins on mobile and have them on desktop (or other way round). This is how it's done:

```javascript live=true fullscreen=true
<Container margin={[0, null, "main"]}>
  <Box sx={{ bg: "mono300", p: 3 }}>Lorem ipsum dolor sit amet</Box>
</Container>
```

You can easily add styles like background or margin to a `Container`:

```javascript live=true fullscreen=true
<Container sx={{ bg: "coral", py: 8 }}>
  <Box>Lorem ipsum dolor sit amet</Box>
</Container>
```

## `Grid`

Let's start with a grid

```javascript live=true fullscreen=true
<Container>
  <Grid>
    {Array.from(Array(32).keys()).map(x => (
      <Box
        key={x}
        sx={{
          p: 7,
          bg: "coral",
          color: "white",
          display: "flex",
          justifyContent: "center",
          alignItems: "center"
        }}
      >
        Test
      </Box>
    ))}
  </Grid>
</Container>
```

`Grid` is a helper around CSS Grid. By default it has 12 equally spaced columns with gutter taken from `theme.gaps.main`.

Let's change number of columns and vertical gap:

```javascript live=true fullscreen=true
<Container>
  <Grid cols={[2, 4, 6, 8, 12]} rowGap={4}>
    {Array.from(Array(32).keys()).map(x => (
      <Box
        key={x}
        sx={{
          p: 7,
          bg: "coral",
          color: "white",
          display: "flex",
          justifyContent: "center",
          alignItems: "center"
        }}
      >
        Test
      </Box>
    ))}
  </Grid>
</Container>
```

Horizontal gap can be custom too:

```javascript live=true fullscreen=true
<Container>
  <Grid gap={[0, null, "1px"]} rowGap={"1px"}>
    {Array.from(Array(32).keys()).map(x => (
      <Box
        key={x}
        sx={{
          p: 7,
          bg: "coral",
          color: "white",
          display: "flex",
          justifyContent: "center",
          alignItems: "center"
        }}
      >
        Test
      </Box>
    ))}
  </Grid>
</Container>
```

Setting CSS grid properties to children:

```javascript live=true fullscreen=true
<Container>
  <Grid>
    {Array.from(Array(32).keys()).map(x => (
      <Box
        key={x}
        sx={{
          p: 7,
          bg: "coral",
          color: "white",
          display: "flex",
          justifyContent: "center",
          alignItems: "center",
          gridColumnStart: ["span 6", null, "span 4", "span 2"] // We set gridColumnSpan for children! Grid has fixed 12 cols.
        }}
      >
        Test
      </Box>
    ))}
  </Grid>
</Container>
```

#### `minItemWidth`

`Grid` comes with a `minItemWidth` property. If used, number of columns will be automatic so that item width won't be lower than `minItemWidth`.

```javascript live=true
<Grid minItemWidth={[100, null, 200]}>
  {Array.from(Array(16).keys()).map(x => (
    <Box
      key={x}
      sx={{
        p: 7,
        bg: "coral",
        color: "white",
        display: "flex",
        justifyContent: "center",
        alignItems: "center"
      }}
    >
      Test
    </Box>
  ))}
</Grid>
```
